Nmedia Sticky Header/Footer is VERY light and simple Plugin which allow to place contents on Top/Bottom with sticky div.

== Description ==
This plugin lets you put sticky header or footer in your website. It can be used to display News Alert, New Features etc on top of your site or at the bottom.
<h2>Features</h2>
<ol>
	<li>Type Custome Message</li>
	<li>HTML supported</li>
	<li>Custom Background Color</li>
	<li>Custom Border Color</li>
	<li>Font Color</li>
	<li>Font Size</li>
	<li>Switch b/w Header and Footer</li>
</ol>

Demo can be seen on company website. You will see sticky footer or header on pages.
<a href="http://www.najeebmedia.com/">View Demo</a>
== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `Setting->Sticky Header/Footer` menu
